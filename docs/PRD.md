## 1. 제품 개요

- **앱 이름**: 핏저니 (FitJourney)
- **슬로건**: "투약과 함께하는 건강한 변화"
- **비전**: GLP-1 투약자가 투약·체중·운동·식사를 한곳에서 기록하고 시각적으로 추적하여 건강한 변화를 직접 체감하는 개인 건강 관리 도구
- **핵심 가치**:
  - 서버 없이 브라우저만으로 동작 (프라이버시 보장)
  - GLP-1 투약 주기 자동 추적 (약품별 간격 자동 적용)
  - 투약-체중-운동-식사 4축 데이터 시각화
  - 목표 설정과 마일스톤으로 동기 부여
  - 다중 사용자 지원 (가족, 지인과 기기 공유)
- **범위 밖(Out of Scope)**: 의료 자문, 처방 관리, 소셜 기능, 서버 동기화, PWA

---

## 2. 타겟 사용자

- **1차**: GLP-1 계열 약품(마운자로/위고비/삭센다/오젬픽) 투약 중인 한국 사용자
- **2차**: 투약 예정자, 투약 완료 후 유지 관리 중인 사용자, 가족과 기기 공유 사용자

| 페르소나 | 설명 | 핵심 니즈 |
|---------|------|----------|
| 김민지 (30대, 직장인) | 마운자로 5mg 투약 3개월차 | 빠른 입력, 모바일 최적화, 진행률 시각화 |
| 이준호 (40대, 운동 병행) | 위고비 + 헬스 병행 | 인바디 데이터, 운동 상세 기록, 통계 |
| 박서연 (50대, 건강 관리) | 삭센다 투약, 식사 조절 중심 | 칼로리 목표, 부작용 추적, 데이터 내보내기 |

## 5. 데이터 모델

### LocalStorage 키 구조 (다중 사용자)

| 키 패턴 | 타입 | 설명 |
|---------|------|------|
| `mj_{userId}_mounjaro` | `Record[]` | 투약 기록 |
| `mj_{userId}_body`     | `Record[]` | 체중/인바디 기록 |
| `mj_{userId}_exercise` | `Record[]` | 운동 기록 |
| `mj_{userId}_diet`     | `Record[]` | 식사 기록 |
| `mj_{userId}_profile`  | `object`   | 사용자 프로필 |
| `mj_{userId}_goals`    | `object`   | 목표 설정 |
| `mj_users`             | `User[]`   | 전체 사용자 목록 |
| `mj_current_user`      | `string`   | 현재 활성 사용자 ID |
| `mj_settings`          | `object`   | 전역 앱 설정 (테마) |
| `mj_version`           | `string`   | 스키마 버전 |

### 레코드 필드 정의

**투약 (`mounjaro`)**
```json
{
  "id": "1710000000000",
  "createdAt": "ISO8601",
  "date": "YYYY-MM-DD",
  "drugName": "mounjaro|wegovy|saxenda|ozempic|other",
  "dose": "5mg",
  "site": "복부|허벅지|팔",
  "sideEffects": ["구역질", "피로감"],
  "cost": 150000,
  "memo": ""
}
```

**체중/인바디 (`body`)**
```json
{
  "date": "YYYY-MM-DD",
  "weight": 75.30,
  "fat": 28.5,
  "muscle": 28.0,
  "bmi": 24.5,
  "water": 35.2,
  "visceral": 8,
  "memo": ""
}
```

**목표 (`goals`)**
```json
{
  "weightStart": 85.0,
  "weightTarget": 70.0,
  "exerciseWeekly": 3,
  "calorieDaily": 1500
}
```

**전역 설정 (`mj_settings`)**
```json
{
  "theme": "dark|light"
}
```
